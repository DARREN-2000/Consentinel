# Product Roadmap

This document outlines the planned trajectory for Consentinel.

## Phase 1: Core Engine & Foundations (Completed)
- ✅ FastAPI Backend with Next-Best-Action Engine
- ✅ In-memory and PostgreSQL database support
- ✅ Integration with base OpenTelemetry & Prometheus
- ✅ React/Vite Dashboard with mock data fallbacks
- ✅ Core AI Agents (Segment, Copy, Journey, Governance)

## Phase 2: Enterprise Integration (Current)
- 🔄 Kafka / Redpanda real-time event streaming integration
- 🔄 Bring-your-own-model (BYOM) framework (Anthropic, self-hosted LLMs)
- 🔄 Webhook endpoints for direct CRM (HubSpot, Salesforce) execution

## Phase 3: Advanced Orchestration
- 📅 Multi-tenant architecture for agency use-cases
- 📅 Custom Fatigue threshold overrides per-audience
- 📅 Native A/B/n experimentation engine

> Note: Features not marked as completed are planned roadmap items and do not currently exist in the codebase.
